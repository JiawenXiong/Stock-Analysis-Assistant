# 股票分析助手

[English](./README_EN.md) | 简体中文

一个 Chrome 浏览器扩展，帮助您在雪球股票页面快速使用 AI 平台进行股票分析。

## 功能特点

- 🔘 **右键菜单触发** - 在雪球股票页面右键即可选择 AI 平台进行分析
- 🤖 **多平台支持** - 支持 Google Gemini、ChatGPT、Kimi、豆包、Grok 等 AI 平台
- ✏️ **自定义提示词** - 可通过弹出界面自定义分析提示词模板
- 📋 **自动填充** - 自动提取股票名称和代码，填充到 AI 平台输入框

## 安装方法

### 从源码安装

1. 下载或克隆本项目到本地
   ```bash
   git clone https://github.com/your-username/stock-analysis-extension.git
   ```

2. 打开 Chrome 浏览器，进入扩展管理页面
   - 地址栏输入：`chrome://extensions/`
   - 或通过菜单：更多工具 → 扩展程序

3. 开启右上角的「开发者模式」

4. 点击「加载已解压的扩展程序」，选择 `stock-analysis-extension` 目录

5. 安装完成，扩展图标将出现在浏览器工具栏

## 使用方法

### 方式一：右键菜单（推荐）

1. 打开任意雪球股票页面，如 `https://xueqiu.com/S/SH600026`
2. 在页面任意位置点击鼠标右键
3. 选择「股票分析」→ 选择目标 AI 平台
4. 扩展将自动打开对应平台并填入分析提示词

### 方式二：点击扩展图标

1. 打开雪球股票页面
2. 点击浏览器工具栏中的扩展图标
3. 选择默认分析平台
4. 点击「分析当前股票」按钮

## 自定义提示词

点击扩展图标打开设置界面，可以自定义分析提示词模板。

### 可用变量

| 变量 | 说明 |
|------|------|
| `{股票名称}` | 当前股票的名称 |
| `{股票代码}` | 当前股票的代码 |

### 默认提示词

扩展默认使用以下提示词模板：

```
帮我从以下维度进行分析股票：

1. 业务理解：用简单术语解释公司业务、解决的问题、付费对象及客户选择它的理由（避免财务术语）。
2. 收入分解：拆解收入流，指出增长/放缓业务及对主要产品/客户的依赖程度。
3. 行业背景：说明所在行业增长/稳定/萎缩状况及长期趋势的影响。
4. 竞争格局：列出主要对手，从定价、产品、规模、护城河对比，突出公司优劣势。
5. 财务质量：分析近年收入增长一致性、利润率、债务、现金流及资本配置。
6. 风险与下跌：识别最大业务/财务/监管风险及可能永久损害业务的因素。
7. 管理层与执行：评估管理团队历史表现及决策对长期股东的影响。
8. 牛熊情景：列出未来3-5年现实牛案与熊案（聚焦基本面，非价格预测）。
9. 估值思考：解释市场如何看待估值，哪些假设最关键及支撑更高/更低估值的条件。
10. 长期论文：形成长期投资论点，总结为什么是好投资、必须成功的条件及出错信号。

最后使用表格汇总；

股票的名称和代码是： {股票名称} {股票代码}
```

## 技术架构

```
stock-analysis-extension/
├── manifest.json          # 扩展配置文件
├── background.js          # Service Worker：右键菜单 + 消息处理
├── content.js             # 雪球页面脚本：提取股票信息
├── content_*.js           # 各 AI 平台脚本：自动填充提示词
├── popup.html/js          # 弹出界面：配置提示词
├── _locales/              # 国际化文件
└── icons/                 # 扩展图标
```

### 核心机制

1. **Storage Bridge 模式**：通过 `chrome.storage.local` 在不同页面间传递数据
2. **Content Script 注入**：向雪球和各 AI 平台页面注入脚本
3. **自动填充**：使用 `document.execCommand('insertText')` 模拟用户输入

## 注意事项

- 本扩展仅在雪球股票页面（`https://xueqiu.com/S/*`）生效
- 部分 AI 平台可能需要登录才能使用
- 如果自动填充失败，提示词会自动复制到剪贴板，可手动粘贴
